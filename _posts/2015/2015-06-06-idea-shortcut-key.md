记录idea的一些常用快捷键

1、进入实现方法的快捷键

> Ctrl + Alt + B 
> > Ctrl + Alt + 鼠标点击

2、自动给方法添加注释的快捷键

> 输入/**,然后回车

3、删除一行

> CTRL + Y / CTRL + X

4、复制一行

> CTRL + D

5、自动导入包

> ALT + Enter

6、生成getter、setter方式

> ALT + insert

7、实现接口方法

> CTRL + O

8、移动整个代码块

> Ctrl + SHIfT + 上下箭头
